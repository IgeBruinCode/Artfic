## Het Platform — drie modules

<a id="p-modules-h2"></a>
**Het Platform** — **H2:** Drie modules. Eén platform.

<a id="p-modules-intro"></a>
Artific bestaat uit drie losstaande modules die met elkaar samenwerken. Gebruik er één, twee of alle drie.

<a id="p-module-assistant"></a>
**Module 01 — AI Assistant** — Je virtuele medewerker, geconfigureerd op maat van je organisatie. Geef de AI Assistant een rol, een persoonlijkheid en toegang tot je kennisbronnen en tools. Hij sluit aan bij een team, volgt je regels en werkt elke dag mee met je mensen. Tags: Interne kennisbank · Klantenservice agent · Verkoopassistent

<a id="p-module-toolbox"></a>
**Module 02 — AI ToolBox** — De self-service AI-laag voor je organisatie. Je IT-team bouwt kleine workflows één keer, en elke medewerker draait ze met één klik. Dagelijkse taken in seconden: een tekst genereren, een document samenvatten, een dataset analyseren. Tags: Dagelijkse taken · Genereren · Samenvatten · Analyseren

<a id="p-module-conversation"></a>
**Module 03 — Conversation Module** — Eén centrale plek voor inkomende e-mail. Koppel Gmail of Microsoft en laat agents binnenkomende berichten lezen, classificeren en een conceptantwoord klaarzetten. Een medewerker reviewt voordat het de deur uit gaat. De snelheid van AI, met het oordeel van je mensen. Tags: Centrale mailbox · Gmail · Microsoft · Human in the loop

## Security & Privacy

<a id="p-security-h2"></a>
**Security & Privacy** — **H2:** Security ingebouwd. Privacy zorgvuldig doordacht.

<a id="p-security-intro"></a>
Artific is ontworpen voor organisaties die controle niet kunnen inruilen voor snelheid. Elke laag van het platform is gebouwd voor enterprise behoeften.

<a id="p-sec-eu"></a>
**Gehost in de EU** — Alle data, alle infrastructuur, alle processing: binnen de EU.

<a id="p-sec-iso"></a>
**ISO 27001 gecertificeerd** — Onafhankelijke audit van ons informatiebeveiligingssysteem. Continu onderhouden.

<a id="p-sec-pseudo"></a>
**Pseudonimisering** — PII wordt gedetecteerd en gepseudonimiseerd voordat het ooit een model bereikt.

<a id="p-sec-model"></a>
**Model-agnostisch** — Wissel GPT, Claude, Gemini of andere modellen per agent, per taak, per tenant. Geen lock-in op één model.

<a id="p-sec-access"></a>
**Access control** — Fijnmazige rollen, scopes en tenant-isolatie. Permissies zijn per agent instelbaar.

<a id="p-sec-audit"></a>
**Audit logs** — Elke prompt, elke tool-call, elke beslissing wordt vastgelegd in het systeem.

## Twee manieren om te bouwen

<a id="p-bouwen-h2"></a>
**Twee manieren om te bouwen** — **H2:** Gebruik onze web portal. Of bouw headless.

<a id="p-bouwen-intro"></a>
Hetzelfde platform eronder. Kies de ervaring die past bij je team.

<a id="p-portal"></a>
**Snelste weg naar productie — Gebruik de Artific Portal** (`portal.artific.app`) — Een gepolijste, white-label web-app voor je medewerkers en admins. Configureer agents in een visuele UI, beheer gebruikers en policies, en rol uit binnen je organisatie, zonder een regel frontend-code.

<a id="p-headless"></a>
**Volledige controle — Bouw Headless** (`agent.ts`-codevoorbeeld met `import { Artific } from '@artific/sdk'`) — Roep de API aan. Embed agents in je product, bouw je eigen frontend, ship naar mobiele apps, plug in op bestaande workflows. Hetzelfde security-model, dezelfde governance, jouw merk en jouw UX.

<a id="p-bouwen-slot"></a>
De meeste klanten starten met de portal en gaan stapsgewijs headless voor delen van hun stack.
