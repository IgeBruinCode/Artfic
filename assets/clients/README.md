# Gedeelde klantlogo-assets

Lokale klantlogo’s voor de bedrijven-slideshows van Brutalistisch A, Conventioneel en Premium. De bestaande slideshows van Minimalistisch en Brutalistisch B behouden hun eigen lokale assetsets.

De bestanden zijn bytegelijk overgenomen uit `brutalistisch-b/assets/clients/`; bronvermelding en herkomst staan in de README van die map.
